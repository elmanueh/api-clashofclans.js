# API Clash of Clans

Este proyecto es una API para interactuar con los datos del juego **Clash of Clans**.

## Configuración

1. Crea un archivo `.env` en la raíz del proyecto.
2. Agrega las siguientes variables de entorno:

```env
API_KEY=
CLAN_TAG=
LINK_API=
API_PORT=

DB_USER=
DB_PASSWORD=
DB_HOST=
DB_DATABASE=
DB_CONNECTION_LIMIT=
DB_WAIT_FOR_CONNECTION=
DB_QUEUE_LIMIT=
```
